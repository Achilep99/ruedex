# Mise à jour RueDex V3 Supabase

Procédure GitHub Desktop propre :

1. Ouvre GitHub Desktop.
2. Fais Fetch origin.
3. Si Pull origin apparaît, fais Pull origin avant de copier les fichiers.
4. Vérifie que tu as 0 changed files.
5. Copie le contenu du ZIP RueDex V3 dans ton dossier ruedex_mvp.
6. Accepte les remplacements.
7. GitHub Desktop doit afficher les fichiers modifiés.
8. Summary : RueDex V3 Supabase
9. Commit to main.
10. Push origin.
11. Va dans GitHub > Actions.
12. Télécharge RueDex-V3-APK quand le workflow est vert.

Important : configure Supabase et les Secrets GitHub avant de tester le mode en ligne. Sans secrets, l'app reste locale.
