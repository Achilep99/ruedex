# Mettre RueDex V2 sur GitHub avec des boutons

1. Décompresse l'archive RueDex V2.
2. Copie **tout le contenu** du dossier décompressé dans ton dossier local actuel `ruedex_mvp` et accepte **Remplacer les fichiers**.
   - Ne supprime pas le dossier caché `.git` de ton projet actuel.
   - Vérifie que le dossier `.github` de la V2 a bien été copié.
3. Ouvre **GitHub Desktop**.
4. Sélectionne le dépôt RueDex.
5. En bas à gauche, écris `RueDex V2` dans Summary.
6. Clique **Commit to main**.
7. Clique **Push origin**.
8. Sur GitHub, ouvre **Actions** → **Construire APK RueDex V2**.
9. Quand tout est vert, télécharge **RueDex-V2-APK** dans Artifacts.

Tu obtiendras :

- `RueDex-developpeur.apk` ;
- `RueDex-utilisateur.apk`.
